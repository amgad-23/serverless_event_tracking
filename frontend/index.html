Serverless Event Tracker

Lightweight serverless event logging and analytics API

API Documentation

GET /

Returns API documentation in HTML (this page).

200 OK
Content-Type: text/html

POST /events

Create a new event. All fields required.

Request Body (JSON): 
{
  "user_id": "123",
  "event_type": "click",
  "timestamp": "2025-07-24T12:00:00Z"
}
Response: 
{
  "message": "Event created"
}

GET /events

List up to 10 most recent events.

Response: 
{
  "events": [
    {
      "user_id": "123",
      "event_type": "click",
      "timestamp": "2025-07-24T12:00:00Z"
    },
    ...
  ]
}

GET /summary

Get event counts. Optional query params: user_id, event_type

GET /summary?user_id=123&event_type=click
Response: 
{
  "total_events": 7
}
